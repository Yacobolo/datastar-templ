# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of datastar-templ
- Type-safe Datastar attribute helpers for templ templates
- Comprehensive event coverage (60+ typed DOM event handlers)
- SSE action builders (Get, Post, Put, Patch, Delete)
- Modifier system (debounce, throttle, once, passive, capture, etc.)
- Signal management (Signals, Computed, Bind)
- DOM manipulation helpers (Text, Show, Class, Style, Attr)
- Merge capability for combining multiple Datastar attributes
- Watchers (OnIntersect, OnInterval, OnSignalPatch)
- Utility functions (Ref, Indicator, Init, Effect, Ignore)
- Full documentation with godoc comments and examples
- Comprehensive test coverage
- CI/CD pipeline with GitHub Actions
- Datastar 1.0.0-RC.7 compatibility
